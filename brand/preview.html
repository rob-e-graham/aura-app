Files in this folder:

  • aura-wordmark-gold.svg, -mono-black.svg, -mono-white.svg — horizontal AURA + sparkle
  • aura-stacked-gold.svg, -mono-black.svg, -mono-white.svg — sparkle above AURA
  • aura-sparkle-gold.svg, -mono-black.svg, -mono-white.svg — sparkle in ring only
  • aura-sparkle-mark-only.svg — sparkle without ring; inherits CSS currentColor

All files are vector SVG — scale infinitely without quality loss. The wordmark SVGs use live <text> referencing Cormorant Garamond from Google Fonts. To export PNG/JPG: open the SVG in a browser → right-click → "Save Image As". For print: open in Illustrator / Sketch / Inkscape and convert text to outlines.
